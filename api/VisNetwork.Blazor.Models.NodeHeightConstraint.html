    If a number is specified, the value is used as the minimum height of the node. The node's height will be set to the minimum if less than the value.

    Valign

    Valid values are 'top', 'middle', and 'bottom'. When specified, if the height of the label text is less than the minimum (including any top or bottom margins), it will be offset vertically to the designated position.
